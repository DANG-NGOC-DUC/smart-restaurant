# Smart Restaurant System

## Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: Supabase(PostgreSQL)
- Auth: Phone + OTP (demo)

## Project Structure

- backend/: API server
- frontend/: React client

## How to run

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
